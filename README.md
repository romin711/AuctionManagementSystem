# Auction Management System

A **console-based Auction Management System built using Core Java** that simulates a simple auction environment. The system allows users to add auction items, register bidders, and perform bidding operations through a command-line interface.

This project was developed to practice **Java programming, Object-Oriented Programming (OOP), and basic system design concepts**. It demonstrates how an auction process can be implemented using core Java structures without external frameworks or graphical interfaces.

## 📌 Features

- Add auction items
- Register bidders
- Start and manage auction bidding
- Set bid limits
- Track the highest bid
- Display auction results
- Simple and interactive console interface

## 📂 Project Structure

The project includes the following main components:

- **AuctionManagementSystem.java** – Main program controlling the auction process  
- **Item class** – Stores item details such as item name, base price, etc.  
- **Bidder class** – Stores bidder information and bidding actions
